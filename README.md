# #MakeoverMonday Contributions

## What is MakeoverMonday?

A weekly data visualization challenge focused on redesigning and improving existing charts.

> MakeoverMonday is a weekly social data project where participants reimagine and improve an existing visualization. Each week features a new dataset and original chart, challenging you to create a clearer, more effective version while practicing your data storytelling skills.

More information available at the [MakeoverMonday Website](https://www.makeovermonday.co.uk/).
